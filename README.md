# Stuff-Plus-Work
These files contain intermediate work and progress in a Stuff+ model for NCAA and Big 10 pitchers using Trackman data. The run_value file calculates run value for different pitch outcomes in order to serve as the target variable. The exploratory_modeling file includes the current exploratory version of the XGBoost model. Both a regression and classification model are included. 
Note: Run-value is calculated with the assistance of 2019 MLB data due to the vast quantity of pitches required. 2019 was chosen as it was the highest run scoring environment in the Statcast era.
